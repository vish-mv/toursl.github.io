QUIZ

created a quiz using html,js and css.my code has several sections.A start button,an information box,a quiz box and a result box
When the user clicks on the "Start Quiz" button, the information box will disappear, and the quiz box will appear.
The quiz box will display a question and a list of options for the user to select. The user will have 15 seconds to select an option, and once they make a selection, they cannot undo it.
If the user does not select an option within 15 seconds, the quiz will move on to the next question automatically. After the user has answered all the questions, the result box will appear and display the user's score.
The user will have the option to replay or quit the quiz.

ABOUT US

created a about us page using html,js,css.Each image is defined with an img tag and includes attributes for height, width, source src, and alternate text alt.
The height and width attributes specify the size of the image, while the source src attribute provides the location of the image file.
The alt attribute provides alternative text for the image, which is displayed if the image cannot be loaded or if the user is using assistive technology such as a screen reader.
